# VidGamify PRO - Gamification Plugin for WordPress

A comprehensive gamification system that adds levels, achievements, leaderboards, streaks, and more to your WordPress site.

## Features

### 🎯 Core Gamification
- **Level System** - XP tracking with automatic level progression
- **Achievements & Badges** - 30+ predefined achievements with rewards
- **Leaderboards** - Global rankings (daily, weekly, monthly)
- **Streaks** - Daily activity tracking with milestone bonuses

### 👥 Social Features
- Followers and friends system
- Groups and clubs with membership fees
- Extended reaction types (like, love, wow, sad...)
- Social statistics dashboard

### 💎 Membership Tiers
- Bronze, Silver, Gold, Platinum levels
- Exclusive benefits per tier
- WooCommerce integration for discounts
- Points-based rewards

### 📊 Analytics & Stats
- Admin analytics dashboard
- Creator performance metrics
- User progress tracking
- Engagement reports

## Installation

1. Upload `vidgamify-pro.zip` to `/wp-content/plugins/`
2. Activate through WordPress admin
3. Configure settings under **Settings → VidGamify PRO**

## Shortcodes

| Shortcode | Description |
|-----------|-------------|
| `[vidgamify_user_level]` | Display user level |
| `[vidgamify_xp_bar]` | XP progress bar |
| `[vidgamify_achievements]` | User achievements |
| `[vidgamify_leaderboard]` | Leaderboard display |
| `[vidgamify_streak]` | Current streak |
| `[vidgamify_membership_tier]` | Membership tier info |
| `[vidgamify_creator_stats]` | Creator statistics |

## Widgets

- XP Progress Widget
- Recent Achievements Widget
- Top Users Leaderboard Widget
- Your Streak Widget
- Social Stats Widget

## Requirements

- WordPress 5.8+
- PHP 7.4+
- MyCred plugin (optional, for points integration)
