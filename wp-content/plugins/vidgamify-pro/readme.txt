A comprehensive gamification system for WordPress with levels, achievements, leaderboards, and more!

== Description ==

VidGamify PRO is a powerful standalone WordPress plugin that adds gamification features to any website. Perfect for video platforms, membership sites, or community-driven content.

= Key Features =

* **Level System** - XP tracking, level progression with bonuses
* **Achievements & Badges** - 30+ default achievements with rewards
* **Leaderboards** - Global and time-based rankings (daily, weekly, monthly)
* **Streaks** - Daily activity tracking with milestone rewards
* **Social Features** - Followers, friends, groups, and clubs
* **Membership Tiers** - Bronze, Silver, Gold, Platinum levels with exclusive benefits
* **Analytics Dashboard** - Track user engagement and progress
* **Creator Stats** - Performance metrics for content creators
* **WooCommerce Integration** - Points conversion and membership discounts
* **Email Notifications** - Keep users engaged with automated alerts

= Requirements =

* WordPress 5.8 or higher
* PHP 7.4 or higher
* MyCred plugin (for points system integration)

= Shortcodes =

* `[vidgamify_user_level]` - Display user level
* `[vidgamify_xp_bar]` - XP progress bar
* `[vidgamify_achievements]` - User achievements
* `[vidgamify_leaderboard]` - Leaderboard display
* `[vidgamify_streak]` - Current streak
* `[vidgamify_membership_tier]` - Membership tier info
* `[vidgamify_creator_stats]` - Creator statistics

= Widgets =

* XP Progress Widget
* Recent Achievements Widget
* Top Users Leaderboard Widget
* Your Streak Widget
* Social Stats Widget

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/vidgamify-pro` or use the WordPress admin "Upload Plugin" feature
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure settings under Settings → VidGamify PRO
4. Use shortcodes and widgets to display gamification elements
